Project: TimeKeeper

Program for monitoring engineering hours spent on projects.



Minimalistic program that utilizes the power of Windows events (lock, unlock, log on, log off, inactivity...) to record hours spent on projects. Most of the time it stays hidden in the taskbar but wakes up and alerts the user on predefined intervals.


  • C#
  • .Net
  • MySQL
  • WPF
  • MVVM
